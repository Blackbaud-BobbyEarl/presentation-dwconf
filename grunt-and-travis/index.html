What is Grunt?

Task-based command line build tool for JavaScript projects.

How to use Grunt

Requirements

npm install -g grunt-cli
npm init -y
npm install grunt --save-dev

Gruntfile.js

module.exports = function (grunt) {
  grunt.initConfig({
    pkg: grunt.file.readJSON('package.json')
  });

  grunt.registerTask('default', function () {
    console.log('Project Name', grunt.config('pkg.name'));
  });
};

What is Travis?

Continuous Integration service used to build and test projects hosted in GitHub.

How to use Travis

Requirements

  • Free for public repos, Signup with your GitHub account
  • Configure through website or via ".travis.yml"
language: node_js
node_js:
  - '0.10'
before_script:
  - npm install -g grunt-cli
  - npm install -g bower
  - bower install
before_install:
  - gem install sass
  - chmod +x update-sky-docs.sh
after_success: ./update-sky-docs.sh
branches:
  only:
    - master
    - /v\d+.\d+.\d+/
cache:
  directories:
    - node_modules
    - bower_components
notifications:
  slack: bb-rdo:asdejiabnoasf
env:
  global:
    - secure: asdflaksjdf;alksjf;lkdfkasdf==
